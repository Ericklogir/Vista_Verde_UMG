# Vista_Verde_UMG - Sistema de Administración de Condominio

## Descripción del Proyecto
Descripción del Proyecto
Sistema de escritorio desarrollado en Java Swing para la administración financiera y operativa de las 30 casas del Condominio Vista Verde. La aplicación permite al administrador gestionar el registro único de propietarios, controlar el cobro de la cuota mensual de Q1,500.00 y generar reportes en tiempo real mediante JTable. Su objetivo principal es automatizar el flujo de caja, detectando de forma inmediata los estados de cuenta y los saldos morosos del mes

## Instrucciones de Ejecución
Para ejecutar este proyecto en cualquier computadora, seguí estos pasos:
1. Asegurate de tener instalado el JDK (versión 25 o superior) y Maven/NetBeans.
2. Cloná el repositorio o descargá el código fuente.
3. Abrí el proyecto en tu IDE preferido.
4. Asegurate de configurar la conexión a la base de datos en el archivo correspondiente.
5. Limpiá, construí (`Clean and Build`) y ejecutá la clase principal.
